

# tool-api

落地页生成工具接口集合

##### 端口:8888

#### 登录注册：

注册接口：

```
路由:/users
示例：http://www.sitkin.top:8888/user
请求方式：post/json
```



| 请求参数 | 说明   | 备注     |
| -------- | ------ | -------- |
| name     | 用户名 | 不可重复 |
| password | 密码   |          |
|          |        |          |

| 返回值 | 说明           | 备注       |
| ------ | -------------- | ---------- |
| code   | 状态码         | 200为成功  |
| data   | 数据库返回信息 |            |
| status | 状态           | true为成功 |
